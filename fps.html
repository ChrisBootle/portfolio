First Person Shooter

C++ & Unreal Engine

Brief:

Your project should feature a clear delineation between C++ code and Blueprint being used in tandem to make a series of player mechanics work in a testing environment. A link between C++ and Blueprint should be managed using property and function specifiers. A communication hierarchy should be established using function calls, interfaces and events to communicate cleanly. Information should be fed back to the player about what is happening using UI, sound and in-world effects managed via C++ hooks into blueprint. Mechanics should be interesting interactions beyond the standard example framework.
