Hangman

C++ - Visual Studio

Brief:

For this assignment, you will develop a text-based game in C++, to showcase your programming skills. The text-based game is your choice but should feature a working game loop and interactivity via command-line input. No graphical output is required for the assignment, this should be a text-based game. Your game should be robust, and cater for invalid input, performing some level of exception handling. Furthermore, the solution should be designed appropriately, using appropriate C++ language constructs to solve the problem.
